== Description ==

QQWorld通行证，支持多种第三方登录，目前支持QQ，微信和微博。

<strong>子付费插件：QQWorld同步器 / QQWorld Synchronizer</strong>

作为QQWorld通行证的扩展插件，将提供各模块的扩展功能。

<ul>
	<li>
		<strong>腾讯QQ</strong>
		<ol>
			<li>暂无</li>
		</ol>
	</li>
	<li>
		<strong>微信</strong>
		<ol>
			<li>自动同步Woocommerce收货地址</li>
			<li>微信自定义菜单</li>
			<li>自动登录</li>
			<li>自动同步文章到微信公众号</li>
			<li>对于未关注公众号的用户，自动显示关注链接</li>
		</ol>
	</li>
	<li>
		<strong>腾讯微博</strong>

== Installation ==

1. 上传 `qqworld-passport` 目录到 `/wp-content/plugins/`
2. 在插件页启用 QQWorld-Passport 插件
